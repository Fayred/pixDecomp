# pixDecomp.py

(decompose image to black and white )

-----

### How decompose an image with ?

![""](assets/image.png)

```bash
$ python3 pixDecomp.py -d -i image.png -f image.txt
```

### How recompose image with ?

!["result1.png"](assets/decomp.png)

